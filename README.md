O Analisador de Sentimentos tem como objetivo analisar frases informadas pelo usuário e identificar
se o sentimento geral da frase é NEGATIVO, NEUTRO ou POSITIVO.

Como funcionalidades atuais e presentes no nosso Analisador nós temos:
A possibilidade do usuário adicionar novas palavras.
A vizualização completa dos dicionários, informando quantidade de palavras e as palavras salvas.
Foi inserido também uma área para o usuário informar o feedback geral do analisador e oque podemos melhorar.

O sistema foi desenvolvido inteiramente em python 3.12.2
